# 📝 AI Content Summarizer

An AI-powered document analysis tool built using **Python, Streamlit, Groq Llama 3.3**, and **Generative AI**. This application allows users to upload PDF documents or paste text, generate summaries, extract key insights, translate content, and interact with documents through an AI-powered chat interface.

---

## 🚀 Features

### 📄 PDF Upload
- Upload PDF documents
- Automatically extract text from PDFs
- Analyze document content using AI

### 📝 Text Summarization
- Generate concise summaries of long content
- Multiple summary lengths:
  - Short
  - Medium
  - Detailed

### 🎯 Key Point Extraction
- Extract important points from documents
- Quickly understand the main ideas

### 🔑 Keyword Extraction
- Identify important keywords and topics
- Useful for research and content analysis

### 🌍 Multilingual Summary
- Generate summaries in:
  - English
  - Hindi
  - Hinglish

### 🤖 Chat With PDF
- Ask questions about uploaded documents
- Get AI-generated answers based on document content
- Interactive document understanding

### 📥 Download Results
- Download generated summaries
- Download key points
- Download extracted keywords
- Download translated summaries

### 📊 Document Statistics
- Word count
- Character count

## 📖 How To Use

### Method 1: Upload PDF

1. Upload a PDF file.
2. Select summary length.
3. Select output language.
4. Click **Generate AI Analysis**.
5. View results in tabs.

### Method 2: Paste Content

1. Paste text into the content box.
2. Select summary length.
3. Select language.
4. Click **Generate AI Analysis**.
